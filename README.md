# IndexAI Protocol

IndexAI is an AI-powered on-chain asset management protocol that autonomously generates and executes optimized DeFi portfolios.

---

## Overview

Managing DeFi portfolios requires constant monitoring, manual rebalancing, and decision-making.

IndexAI removes this complexity by using AI to generate portfolio allocations and execute them directly on-chain via smart contracts.

---

## What It Does

- Generates AI-driven portfolio allocations  
- Executes trades on-chain  
- Stores portfolio state on-chain  
- Enables automated portfolio management  

---

## How It Works

1. User connects wallet  
2. AI analyzes market conditions  
3. AI generates optimal allocation  
4. Smart contract executes transactions  
5. Portfolio is recorded on-chain  

---

## Architecture

- **Frontend** → Next.js  
- **Backend** → Node.js (AI + API)  
- **Smart Contracts** → Solidity (Hardhat)
